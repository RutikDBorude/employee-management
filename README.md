# Employee Management System

Simple Java web application using Servlets, JSP, JDBC, and MySQL.

Features:
- Add / View / Edit / Delete employees
- MVC architecture (Servlets + JSP + DAO)
- MySQL database integration

How to run:
1. Configure `DBUtil.java` with your MySQL credentials.
2. Run: `mvn clean package tomcat7:run`
3. Open: `http://localhost:8080/employee-management/employees`
